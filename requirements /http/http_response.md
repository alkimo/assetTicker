# Http Response

## Main Flow
1. ✅ Valid Request, GET HTTP request 
2. ✅ Valid Request, POST HTTP request
3. Ok - 200, Get Data Back
4. No Content - 204, No Data Back

## Errors
1. Bad Request - 400
2. Unauthorized - 401
3. Forbidden - 403
4. Not Found - 404
5. Internal Server Error - 500

## Exception, Different Status Code 
1. Internal Server Error - 500

## Exception, Invalid HTTP Verb
1. Internal Server Error - 500

## Exception, HTTP Request Returns Some Exception 
1. Internal Server Error - 500
